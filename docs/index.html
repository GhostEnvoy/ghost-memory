How does semantic search work?

Traditional databases find exact matches. Ghost Memory converts your question into a "vector" ‚Äî a list of numbers that capture meaning. Then it finds the closest matches in the database.

"coffee" ‚âà [0.12, -0.34, 0.56...] ‚Üí finds "espresso", "latte", "caffeine"

What is vector embedding?

Embedding converts text into lists of numbers. Words with similar meanings have similar numbers. This is what enables "fuzzy" semantic search.

"coffee"  ‚Üí [0.12, -0.34, 0.56, -0.89, ...]
"espresso" ‚Üí [0.11, -0.33, 0.55, -0.88, ...]  ‚Üê similar!
What's Qdrant?

Qdrant is a vector database ‚Äî optimized for storing and searching embeddings. It's the engine that makes semantic search fast and scalable.
